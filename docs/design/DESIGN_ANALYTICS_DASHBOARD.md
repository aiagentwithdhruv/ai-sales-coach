# Design Document: Analytics Dashboard

**Feature:** Analytics Dashboard  
**Priority:** P1 (Phase 1)  
**Designer:** AI Sales Coach Team  
**Last Updated:** 2026-02-01  

---

## 1. Overview

### 1.1 Purpose
Provide sales reps and managers with visibility into skill progression, practice habits, and improvement opportunities through data visualization.

### 1.2 User Goals
- Track personal improvement over time
- Identify specific skill gaps
- Stay motivated with streaks and achievements
- Managers: Monitor team performance and coaching priorities

### 1.3 Design Principles
- **Glanceable**: Key metrics visible immediately
- **Actionable**: Every insight links to an action
- **Motivating**: Celebrate progress and streaks
- **Honest**: Don't hide poor performance

---

## 2. Dashboard Variants

### 2.1 Individual Dashboard (Rep View)
Focus on personal progress, streaks, and improvement areas.

### 2.2 Team Dashboard (Manager View)
Focus on team comparisons, coaching priorities, and aggregate trends.

## 8. Responsive Behavior

### Desktop (1024px+)
- 2-column layout for charts
- Full leaderboard visible
- Heatmap calendar visible

### Tablet (768px - 1023px)
- Single column layout
- Charts stack vertically
- Leaderboard shows top 5

### Mobile (< 768px)
- Stats as horizontal scroll
- Charts simplified (7 days max)
- Leaderboard as cards
- No heatmap (replaced with streak display)

---

## 9. Accessibility

- Charts have aria-labels describing data
- Color not sole indicator (icons + text)
- Keyboard navigation for time selector
- Focus states on all interactive elements
- Screen reader friendly table structure for leaderboard
